## Создание глобальных атрибутов 

В X-Cart есть два способа создания глобальных атрибутов.

**Способ 1:**

Данный способ предусматривает создание глобальных атрибутов в разделе **Классы и атрибуты** (**Каталог / Классы и атрибуты**).


1.  В панели управления магазина откройте страницу **Каталог / Классы и атрибуты**.
2.  Нажмите ссылку **Редактировать атрибуты** справа от заголовка **Глобальные атрибуты**:
    ![1.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/1.jpg)
    Откроется страница редактирования глобальных атрибутов:
    ![2.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/2.jpg)
3.  Нажмите **Новый атрибут**:
    ![3.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/3.jpg)
    Откроется окно для создания нового атрибута:
    ![4.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/4.jpg)
4.  Внесите информацию о новом атрибуте:

    *   **Атрибут**: название нового атрибута.
    *   **Группа атрибута**: название группы атрибута (указывается, если новый атрибут должен быть присвоен какой-то группе атрибутов).
    *   **Тип**: тип значения атрибута (предзаполненное поле, текстовое поле, да/нет).![5.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/5.jpg)
5.  Нажмите **Далее**. Атрибут создан. В этом же окне можно редактировать его значения:
    ![6.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/6.jpg)
6.  Настройте значения атрибута (дополнительная информация: {% link "Работа со значениями атрибутов" ref_SuWz9rmN %}). В качестве примера, зададим значения для атрибута _Цвет_, для этого нужно обозначить цвета всех товаров, которым будет присвоен этот глобальный атрибут: 
    ![7.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/7.jpg)
7.  Чтобы несколько или все значения нового атрибута автоматически присваивались всем товарам, которые будут добавлены в каталог магазина в будущем, нажмите галочки справа от значений атрибута (галочки зеленого цвета, когда активированы): 
    ![8.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/8.jpg)
8.  Нажмите **Сохранить**:
    ![9.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/9.jpg)
    Значения атрибута сохранены. 

    Создан новый атрибут. Если на шаге 7 для всех его значений вы поставили галочки, эти значения будут автоматически присвоены новым товарам, которые будут созданы в магазине. Новый атрибут не применяется автоматически к товарам, которые уже существовали в магазине на момент создания атрибута. Значения нового атрибута можно присвоить существующим товарам вручную на странице **Атрибуты** каждого товара. Редактируя один товар, вы можете включить опцию **Применить изменения значений атрибутов для всех товаров**, чтобы применить такие же изменения ко всем товарам магазина.

**Способ 2:**

Данный способ предусматривает редактирование товара и создание глобальных атрибутов в разделе **Атрибуты** этого товара. 


1.  В панели управления магазина откройте страницу любого товара, для которого нужен новый атрибут, и перейдите на вкладку **Атрибуты**:
    ![10.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/10.jpg)
    Откроется страница редактирования атрибутов этого товара. 
2.  Пройдите по ссылке **Глобальные атрибуты**. Это раздел, где вы можете создать глобальные атрибуты и атрибуты класса товаров для данного товара:
    ![11.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/11.jpg)
    В секции **Глобальные атрибуты** создайте новые атрибуты, которые будут действовать на уровне всего магазина:
    ![12.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/12.jpg)

3.  Укажите данные новых атрибутов:
    1.  В меню **Добавить атрибут** выберите тип нового атрибута (предзаполненное поле, текстовое поле , да/нет). В списке атрибутов появится новая пустая строка, в которой необходимо указать название нового атрибута и его значения. 
        ![13.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/13.jpg)
        (Дополнительная информация о типах значений атрибутов: {% link "Работа со значениями атрибутов" ref_SuWz9rmN %}). 
        Для примера, мы создадим атрибут _Цвет_ со значениями _Серый, Белый_ и т.д., поэтому нам нужен тип атрибута _Предзаполненное поле_. Тип атрибута _Предзаполненное поле_ - это значение по умолчанию, поэтому можно просто нажать на кнопку **Добавить атрибут** и не выбирать значение из меню:
        ![14.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/14.jpg)

    2.  Укажите название и значения атрибута:
        ![15.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/15.jpg)

    3.  Укажите область применения атрибута:

        *   Для всех товаров в каталоге.
        *   Только для редактируемого товара. Изменение названия атрибута и списка атрибутов (например, изменение позиции атрибута в списке или удаление атрибута из списка) затрагивает все товары в магазине. Например, если значение _Серый_ атрибута _Цвет_ применено глобально, в описание всех товаров в магазине появится атрибут _Цвет - Серый_. 
        ![16.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/16.jpg)
        Обратите внимание, глобальное применение атрибута распространяется только на товары, существующие в магазине на момент совершения данных изменений. Атрибут не будет автоматически присвоен товарам, которые будут созданы в каталоге в будущем.
    4.  Нажмите **Сохранить**:
        ![17.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/17.jpg)

        Атрибут создан:
        ![18.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/18.jpg)

        В разделе **Характеристики** товара в магазине появился атрибут _Цвет - Серый_. Если на шаге 4 это значение атрибута было применено глобально, то у всех товаров появилась такая же запись. Если значение атрибута не было применено глобально, запись появилась только у редактируемого товара. Но для других товаров это значение атрибута всегда можно задать вручную.

## Просмотр глобальных атрибутов магазина


1.  В панели управления магазина откройте странницу **Каталог / Классы и атрибуты**.
2.  Нажмите **Редактировать атрибуты** справа от заголовка **Глобальные атрибуты**:
    ![20.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/20.jpg)
    Откроется страница, содержащая список глобальных атрибутов всех товаров магазина:
    ![21.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/21.jpg)

Для каждого атрибута в списке указано его название и тип значения. 

Как посмотреть значения атрибута:

1.  Наведите курсор на название атрибута. Справа появится кнопка **Изменить**:
    ![22.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/22.jpg)
2.  Нажмите **Изменить**. Откроется окно с информацией об атрибуте и его значениях:
    ![23.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/23.jpg)

## Присвоение глобального атрибута товару

Когда глобальные атрибуты уже созданы (на странице **Каталог / Классы и атрибуты**), они должны быть присвоены товарам.

В качестве примера, глобальный атрибут _Цвет - Желтый_ будет присвоен товару _Сумка_:

1.  В панели управления магазина найдите товар, для которого нужно создать глобальный атрибут, и перейдите на вкладку **Атрибуты**:
    ![24.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/24.jpg)
    Откроется страница редактирования атрибутов товара. 
2.  Перейдите в раздел **Глобальные атрибуты**, здесь для данного товара можно задать глобальные атрибуты, а также, атрибуты класса товаров:
    ![25.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/25.jpg)

3.  В секции Глобальные атрибуты перечислены все глобальные атрибуты магазина. Если у какого-то атрибута не указано значение, значит, он еще не применен ни к какому товару и не доступен в магазине:
    ![26.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/26.jpg)
     
4.  Как присвоить атрибут товару:
    *   впишите значение атрибута вручную:
    *   или нажмите на кнопку меню поля значения атрибута и выберите одно из готовых значений: 
        ![27.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/27.jpg)
        В качестве примера, мы установили значение _Желтый_ атрибута _Цвет_:
        ![28.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/28.jpg)
        Таким же образом можно присвоить несколько атрибутов одному товару.

5.  Включите или отключи те опцию **Применить изменения значений атрибутов для всех товаров**, чтобы применить изменение ко всем товарам магазина или только к редактируемому.
    ![29.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/29.jpg)

6.  Нажмите **Сохранить**.
    ![30.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/30.jpg)
    Атрибут и его выбранное значение присвоены товару и доступны в магазине.

## Редактирование глобальных атрибутов

Как изменить название и свойства глобального атрибута:

**Способ 1:**

1.  В панели управления магазина откройте страницу **Каталог / Классы и атрибуты**.
2.  Нажмите **Редактировать атрибуты** справа от заголовка **Глобальные атрибуты**:
    ![32.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/32.jpg)
    Откроется страница редактирования глобальных атрибутов магазина.
3.  В списке атрибутов найдите тот, который необходимо изменить и наведите на него курсор. Справа появится кнопка **Изменить**:
    ![33.jpg]({{site.baseurl}}/attachments/ref_o3s6sQgq/33.jpg)
4.  Если нужно изменить только название атрибута, нажмите внутри поля с названием и впишите новое. Если нужно изменить значения атрибута (или название и значения), нажмите **Изменить**, откроется окно, в котором можно редактировать всю информацию об атрибуте.
5.  Некоторые или все значения атрибута могут автоматически присваиваться всем товарам, которые будут созданы в каталоге в будущем. Для этого справа от значений атрибута нужно поставить галочки.
6.  Нажмите **Сохранить**.

**Способ 2:**

1.  В панели управления магазина найдите товар, для которого нужно редактировать глобальный атрибут, и перейдите на вкладку **Атрибуты**. Это может быть любой товар с таким атрибутом, т.к. изменения для одного товара можно автоматически применить ко всем остальным товарам с таким же атрибутом.
2.  На странице **Атрибуты** перейдите в **Глобальные атрибуты**.
3.  На открывшейся странице найдите секцию **Глобальные атрибуты**.
4.  Найдите атрибут, который нужно изменить:

    *   Наведите курсор на атрибут, нажмте на название и редактируйте поле. 
    *   Измените значения атрибута в полях значений.

5.  Включите или отключите опцию **Применить изменения значений атрибутов для всех товаров**:

    *   Опция включена => Изменеяются все товары в магазине.
    *   Опция выключена => Изменяется только редактируемый товар. Изменение названия атрибута и списка атрибутов (например, изменение позиции атрибута в списке или удаление атрибута из списка) затрагивает все товары в магазине.
6.  Нажмите **Сохранить**.

## Удаление глобальных атрибутов


**Способ 1:**

1.  В панели управления магазина откройте страницу **Каталог / Классы и атрибуты**.
2.  Нажмите **Редактировать атрибуты** справа от заголовка **Глобальные атрибуты**.
Откроется страница редактирования глобальных атрпибутов магазина.
3.  В списке атрибутов найдите тот, который необходимо удалить, и нажмите на значок корзины справа от него.
4.  Нажмите **Сохранить**. Атрибут удален из списка глобальных атрибутов магазина и из описаний товаров в магазине.

**Способ 2:**

1.  В панели управления магазина найдите любой товар с глобальным атрибутом, которй нужно удалить.
2.  На странице **Атрибуты** этого товара перейдите в **Глобальные атрибуты**.
3.  На открывшейся странице найдите секцию **Глобальные атрибуты**.
4.  В списке найдите атрибут, ктотрый нужно удалить, и нажмите на значок корзины справа от него.
5.  При этом, статус опции **Применить изменения значений атрибутов для всех товаров**
 не имеет значения, т.к. удаление глобального атрибута затрагивает все товары в магазине. Глобальный атрибут будет удален из списка глоальных атрибутов и из описаний всех товаров.
6.  Нажмите **Сохранить**.
